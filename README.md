## Deploy to GitHub Pages
1. Create a new public repo at github.com/RohitMGothwal (e.g. name it `portfolio`).
2. Upload `index.html` to the repo (Add file → Upload files → Commit).
3. Go to Settings → Pages → Source → Deploy from a branch → `main` / `/ (root)` → Save.
4. Your site goes live at `https://RohitMGothwal.github.io/portfolio/` within about a minute.
